# Contexto

A aplicação agora já está minimamente funcional, a ideia dessa tarefa agora é melhorar alguns aspectos do front-end, como design e melhoria na implementação do código.

# Melhorias

É necessário implementar uma interface moderna seguindo melhores padrões de UX, seguindo a paleta de cores da Netflix.

A aplicação deve ter 3 páginas:

- Central de upload, onde é possível fazer upload de vídeos e ver o status de upload dos vídeos atuais
- Catálogo simples de vídeos, onde ao clicar em qualquer um deve ser redirecionado para o player do vídeo
- Player de Vídeo otimizado e com experiência de usuário melhor.

Utilize skills que possam ajudar:

- frontend-design
- context7-mcp
- vercel-react-best-practices
- tanstack-start-best-practices
- shadcn

É **OBRIGATÓRIO** implementar o ShakaPlayer como vídeo, utilizar shadcn para estilização e React Query (tanstack) para integração com as APIs.
